
简介
---------------

Mantoui 是构建在 OmniRom 基础上的一个小版本；

在此请首先对 OmniRom 的贡献者表示感谢；

Mantoui 仅凭个人爱好，不喜者勿扰 . 谢谢！


版本
---------------

Android：4.4.4 r1

OmniRom: android-4.4


下载
---------------

获取:

    repo init -u git://github.com/mantoui/m_manifests.git -b Kitkat

同步:

    repo sync

编译:

     cd <source-dir>; . build/envsetup.sh; brunch <device_name>
